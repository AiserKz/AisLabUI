# UI Kit

Source-based UI kit built on top of Tailwind CSS and daisyUI.

## Requirements

- Tailwind CSS
- daisyUI

## Usage

Add native components:
```bash
npx @aislab/ui-kit native
````

Add motion components:

```bash
npx @aislab/ui-kit motion
```

Components will be copied into:

```
src/components/ui/
```

You fully own the code and can customize it.
